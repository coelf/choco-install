# Script to install some app with chocolatey

## How to
on a powershell or cmd with admin rights

```
install-app.bat
```

it will install  chocolatey and the app from the file install-app.ps1


## Tips choco
### find app
```
choco search <app>
```

### install app
```
choco install <app> (-y)
```

### update all application managed by chocolatey
```
cup all -y
```
